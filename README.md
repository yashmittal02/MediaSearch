# 🔍 MediaSearch

A modern media search application built with **React**, **Redux Toolkit**, and **Tailwind CSS** that allows users to search **Photos**, **Videos**, and **GIFs** from multiple APIs in one place.

---

## 🚀 Features

- 🔎 Search Photos, Videos, and GIFs from a single interface
- 📷 High-quality Photos powered by Unsplash API
- 🎥 HD Videos powered by Pexels API
- 🎉 Animated GIFs powered by GIPHY API
- ⚡ Global state management using Redux Toolkit
- ❤️ Save your favorite media to a personal Collection
- 💾 Persistent Collections using Local Storage
- 🔥 Trending Search Suggestions
- 📱 Fully Responsive Design
- 🎨 Modern UI built with Tailwind CSS
- 🔔 Toast Notifications for user actions
- 🚀 Fast API integration using Axios
